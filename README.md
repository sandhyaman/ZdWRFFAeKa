# Python招聘数据爬取可视化系统

## 核心功能
- 数据爬取：使用Python语言和selenium爬虫技术，模拟登录BOSS直聘网站，抓取岗位信息。
- 数据提取：利用beautifulsoup库解析网页源码，提取工作地点、薪资范围、学历要求、工作经验、公司性质等字段。
- 数据存储：将提取的数据存储到MySQL数据库中，方便后续分析和处理。
- 数据可视化：采用pyecharts库，将数据以柱状图或饼状图等形式进行可视化展示。
- 分析报告：根据爬取和可视化的结果，生成包含1.6万字论文和PPT的分析报告。

## 适用场景
- 招聘市场分析：帮助企业了解当前招聘市场的趋势和特点。
- 岗位信息对比：为求职者提供不同岗位的对比分析，帮助其选择合适的工作。
- 人才需求预测：根据招聘数据，预测未来的人才需求趋势。

## 技术特点
- 使用Python语言进行开发，具有高效性和易用性。
- 采用Django后端框架，保证系统的稳定性和安全性。
- 使用MySQL数据库存储数据，确保数据的安全性和可靠性。
- 利用pyecharts进行数据可视化，直观展示数据趋势。
- 使用selenium进行爬虫，有效模拟真实用户行为。
